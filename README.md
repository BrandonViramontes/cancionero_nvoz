## Convención de commits

Para mantener un historial claro, usar estos prefijos:

Tipos específicos para cancionero:


- `song:` agregar nueva canción
- `lyrics:` correcciones ortográficas o modificacion de la letra
- `chords:` ajustar acordes o tablaturas
- `structure:` añadir o reorganizar canciones o secciones
- `config:` cambios de diseño, paquetes LaTeX, preámbulo o configuración
- `notes:` cambios en anotaciones (\textnote), comentarios (li={}) o documentación auxiliar.
- `metadata:` actualizar información de canciones (autor, nombre del canto, sr)
- `build:` modificar compilación LaTeX (latexmkrc)
